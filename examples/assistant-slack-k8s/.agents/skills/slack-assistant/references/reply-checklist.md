# Reply checklist

Before posting with `reply_in_slack_thread`, make sure your reply:

- Answers the actual question, with the leading `<@…>` mention token ignored.
- Is concise and free of filler openers ("Sure!", "Great question").
- States plainly when you don't know, and asks at most one focused clarifying
  question instead of guessing.
- Uses Slack `mrkdwn` only where it helps (`*bold*`, `` `code` ``, bullets); no
  walls of formatted text.
- Is a single message — one reply per mention, no unsolicited follow-ups.
